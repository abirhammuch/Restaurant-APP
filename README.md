# 🍽️ Restaurant App

A modern and responsive full-stack restaurant web application built with React and Node.js. Users can browse delicious meals, explore food categories, and enjoy a clean and interactive user interface.

## 🌐 Live Demo

https://restaurant-app-marshal-nine.vercel.app/

## 📂 Repository

https://github.com/abirhammuch/Restaurant-APP

## ✨ Features

- Responsive design
- Modern user interface
- Browse restaurant menu
- Food category filtering
- Beautiful food cards
- Fast and interactive experience
- Frontend and backend separated

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- JavaScript
- Vite

### Backend
- Node.js
- Express.js

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/abirhammuch/Restaurant-APP.git
```

### Navigate to the project

```bash
cd Restaurant-APP
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

### Install Backend

```bash
cd backend
npm install
npm start
```
